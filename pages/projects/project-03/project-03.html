React Native Jobs

A job search app developed with React Native and Expo using JavaScript. It uses the Jsearch API for the jobs source.

Test the app with Expo Go:

  1. Download the Expo Go app from Apple App Store or Google Play Store
  2. Scan the QR code that's shown in this link
  3. After you scan the QR code you should be able to test the app through Expo Go
